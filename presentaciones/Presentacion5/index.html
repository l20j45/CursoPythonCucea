For
Ya vimos la sentencia mientras, ahora existe la sentencia desde que nos ayuda a nosotros saber cuantas veces se va a repetir un codigo
para programar esto es asi 


                    for "elemento" in "iterable" : 
                        (identacion) "codigo"
Cosa interesante, en python para la iteracion se define un elemento o rango
esto es a diferencia de los otros lenguajes de programacion
para definir un rango basico podemos usar el siguiente codigo
table = range(1, 11)
los tipos de datos range corresponden a tipos de datos especiales que contienen una serie de numeros
la forma de crear una es la siguiente range ("numero iniciar","numero final","pasos")
for no solo recorre rangos tambien puede recorrer objetos como diccionarios, listas, pilas, colas arreglos pero estos datos los veremos mas adelante

Funciones
esa no es la unica forma de repetir codigos, tambien existen las funciones
Que son pequeños segmentos de codigo que pueden ser reutilizados varias veces
la forma de utilizarlas seria la siguiente 

                    def "nombre de la funcion":
                        (identacion)"codigo"
piensa que a la mejor siempre quieres limpiar pantalla, un separador y un saludo
como lo harias?
funciones con parametros
las funciones pueden recibir valores para que ejecuten alguna opcion esto se llama parametros
la forma de usarlas es la siguiente


                    def "nombre de la funcion ("variable","variable",.....):
                        (identacion)"codigo"
Puedes pasarle cualquier tipo de valor y hacer que en el funcione un miniprograma para que haga cierta opcion
supongamos entonces que nuestro programa de venta de manzanas, ahora vende duraznos y peras como podriamos hacerlo
porque pasamos las variables como valores? hay algo que se conoce como Variable scope (ambito)
esto nos dice que las variables solo existen en donde fueron declaradas
esto quiere decir que si tu usas una variable en el programa principal, solo funciona en esa parte
y si creas una variable en alguna funcion solo funciona en esa parte
ejemplo
pero siguiente esta idea, con el scope nos dice que las variables se quedan dentro de la funcion
como podemos sacarlas?
funciones con retorno de variable
este tipo de funciones generan su comportamiento, y despues nos devuelven x valor para su uso posterior
la forma de utilizarlas es la siguiente


                def "nombre de la funcion ("variable","variable",.....):
                    (identacion)"codigo"

                    return "variable a regresar"
este tipo de funciones son las mas utiles pues hacen todo su proceso y nos permiten trabajar con los resultados
son una forma de limpiar nuestro codigo, y nos permiten modularizar el codigo
